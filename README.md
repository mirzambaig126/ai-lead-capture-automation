# AI Lead Capture Automation

## Project Overview

This project automates the lead capture and follow-up process using Make.com and AI. When a lead submits a Google Form, the workflow generates a personalized email using the Groq API and automatically sends it to the lead while storing all information in Google Sheets.

---

## Business Problem

Many businesses manually respond to new leads, resulting in slower response times and inconsistent communication.

This automation reduces manual work and ensures every lead receives a personalized response within seconds.

---

## Workflow

Google Form
➡️
Make.com
➡️
Groq AI API
➡️
Generate Personalized Email
➡️
Gmail Sends Email
➡️
Google Sheets Stores Lead
